## Project Info
**FreshScanX** is a modern web application designed to help users verify the freshness of fruit. By simply pointing your camera or uploading an image, our advanced freshness checker model instantly analyzes the produce to determine if it is fresh or rotten.

## Getting Started

### Prerequisites
- Node.js & npm installed

### Installation
1. Clone the repository:
   ```sh
   git clone <YOUR_GIT_URL>
   cd FreshScanX
   ```

2. Install dependencies:
   ```sh
   npm install
   ```

3. Start the development server:
   ```sh
   npm run dev
   ```
